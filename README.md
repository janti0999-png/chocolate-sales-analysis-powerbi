# Chocolate Sales Analysis Dashboard (Power BI)

## Overview
This project analyzes chocolate sales data to identify revenue trends, top-performing products, and regional performance.

## Tools Used
- Power BI
- Excel

## Key KPIs
- Total Revenue
- Top Products
- Sales by Region
- Monthly Sales Trends

## Key Insights
- A few products contribute significantly to total revenue
- Sales trends vary across different regions
- Seasonal patterns impact overall performance

## Features
- Interactive dashboard with slicers
- KPI cards for quick overview
- Trend analysis using charts

## Files Included
- Power BI Dashboard (.pbix)
- Dataset (.xlsx/.csv)
- Dashboard screenshot
